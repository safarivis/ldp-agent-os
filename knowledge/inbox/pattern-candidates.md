# Pattern Candidates

Use this file as the lightweight queue for possible reusable patterns spotted during project work.

## How to use
- Add short notes when something looks reusable across projects.
- Keep it simple.
- Promote confirmed patterns later into `knowledge/workflows/`, `knowledge/tools/`, or `templates/`.

## Standard review prompt
At the end of major tasks, ask:
- "Anything here we should standardize or reuse?"
- or shorter: "Any patterns?"

## Template

### YYYY-MM-DD — <short pattern name>
- Source: <project / repo / cell>
- Why it may be reusable: <one line>
- Suggested destination: <workflow / tool note / template / stay local>
- Status: candidate
