# Prompt Library

This directory stores reusable prompt templates for the RevvTech AgentOS.

## Purpose

Keep proven prompts separate from live client/project context.

## Rules

- Store reusable prompts here.
- Keep active project notes in `workspace/`, not here.
- Start prompts as `draft` until they are proven.
- Promote only prompts that work repeatedly.

## Naming

Use:

`[topic]__prompt__[status]__v[version].md`
