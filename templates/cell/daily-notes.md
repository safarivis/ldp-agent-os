# Daily Notes

Use this file as the running day-by-day log for this cell.

## Rules

- Add short dated entries in reverse-chronological order.
- Capture what was done, what was sent, what changed, and what is waiting.
- Keep it concise and operational.
- Put durable decisions in `decisions.md`.
- Put formal message summaries in `email-log.md`.
- Put larger topic summaries in `notes.md` or `project-notes.md`.

## Template

Use this lightweight pattern for new entries:

```md
## YYYY-MM-DD

- Done:
  - ...
- Sent:
  - ...
- Changed:
  - ...
- Waiting on:
  - ...
- Next:
  - ...
```

Use only the sections that matter for the day.
